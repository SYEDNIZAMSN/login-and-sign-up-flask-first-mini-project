# Flask Login & Signup System

This is a simple **Flask-based Login and Signup System** created by **Syed Nizam**.  
The project demonstrates:

- Radio button selection (`Login` or `Signup`)
- Dynamic form rendering based on user choice
- Session handling
- Login validation
- Signup check
- Logout functionality
- Welcome message after successful login/signup
